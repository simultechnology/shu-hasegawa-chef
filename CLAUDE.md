# Shu Hasegawa - Executive Sushi Chef Website

## Project Overview
Shu Hasegawaさん（フランス・パリの寿司職人エグゼクティブシェフ）の個人Webサイト。

## Tech Stack
- HTML5 / CSS3 / Vanilla JavaScript
- Google Fonts (Cormorant Garamond, Zen Kaku Gothic New)
- GitHub Pages でホスティング

## Design Concept
- Japanese Wabi-Sabi meets Parisian Elegance
- カラー: 漆黒ベース + 金箔アクセント + 朱色ハイライト

## File Structure
```
├── index.html      # メインページ
├── styles.css      # スタイリング
├── script.js       # インタラクティブ機能
└── assets/images/  # 画像素材
```
